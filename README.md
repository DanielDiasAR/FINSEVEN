O projeto FinSeven é uma plataforma web que está sendo desenvolvida com o objetivo de ter um ambiente em que os usuários possam realizar seu cadastro e ter o controle de suas finanças pessoais como despesas, 
receitas, investimentos, emitir relatórios e visualizar gráficos em tempo real referente aos saldos e valores. Futuramente vamos incluir várias funcionalidades como metas a alcançar, 
integração com IA para obter as melhores sugestões sobre investimentos e muito mais.
